# ДЕРЕВО ЖИТТЯ 🌳 - Люди Утворюють Форму

Революційна гра-арт де **люди (точки) самі створюють форму дерева** через притягання до невидимої архітектури!

## 🎨 Ідея

```
🌳 НЕВИДИМЕ ДЕРЕВО (архітектура у просторі)
        ↓
    👥👥👥 ЛЮДИ (точки) починають падати з неба
        ↓
    ⚡ Люди притягуються до вузлів цього невидимого дерева
        ↓
    🎯 Вони природньо рухаються до гілок, листків, кореня
        ↓
    ✨ ЛЮДИ УТВОРЮЮТЬ ВИДИМУ ФОРМУ ДЕРЕВА!
        ↓
    = ЖИВЕ ДЕРЕВО ІЗ ЛЮДЕЙ =
```

## 🌳 Архітектура Дерева

### Структура (невидима, але магічна):

```
        🍃🍃🍃  ЛИСТКИ (верхня частина)
        / | \
      🌿 🌿 🌿  ГІЛОЧКИ (розходяться)
       \ | /
        🌳    ГІЛКИ (головні)
        |
        |      СТОВБУР (центр)
        |
        📌    КОРІННЯ (внизу)
```

### Вузли дерева:
- **Коріння** (внизу, глибина 0) - сильне притягання
- **Стовбур** (посередині, глибина 1) - центр
- **Великі гілки** (розходяться, глибина 2)
- **Малі гілочки** (розходяться далі, глибина 3)
- **Листки** (верхня частина, глибина 4)

## 🎮 Механіка

### Люди (Точки)
- 🌈 Кожна має унікальний колір
- 👥 Народжуються далеко у космосі
- ⚡ **Притягуються до вузлів дерева** (головна сила!)
- 🤝 Трохи притягуються одна до одної
- 🟢 Лінії з'єднують близькі людей (утворюючи видиму структуру)

### Результат
- Люди мігрують до дерева
- Вони натуральне розташовуються по гілках
- Утворюють ВИДИМУ форму дерева!
- Дерево ЖИВЕ через людей!

### Метафора для України 🇺🇦

```
🇺🇦 НЕВИДИМЕ ДЕРЕВО = ДУХ НАЦІЇ
   (історія, культура, мова, цінності)

👥 ЛЮДИ = ІНДИВІДУАЛЬНІ УКРАЇНЦІ
   (з різних регіонів, міст, сімей)

⚡ ПРИТЯГАННЯ = ПРИВ'ЯЗАНІСТЬ ДО ОДКОРІННЯМ
   (люди чують зв'язок з корінням)

🌳 ВИДИМЕ ДЕРЕВО = УКРАЇНСЬКА ЄДНІСТЬ
   (коли люди рухаються разом, утворюють націю)
```

## 🎯 Чому це красиво

1. **Органічність** - точки природньо знаходять свої місця
2. **Фрактальність** - структура повторюється на кожному рівні
3. **Емерджентність** - цілісна форма виникає з простих правил
4. **Рух** - дерево живе, дише, росте з людьми

## 🌟 Технічні детали

### Гравітація
```
Люди притягуються до вузлів дерева:
F = (strength * 100) / (distance ^ 1.5)

strength залежить від глибини вузла
(глибші вузли сильніші)
```

### Структура вузлів
- **Коріння** - 1 вузол (x=0, y=200)
- **Стовбур** - 1 вузол (x=0, y=0)
- **Гілки** - 2 вузли (розходяться вгору-вниз)
- **Гілочки** - 4 вузли (розходяться далі)
- **Листки** - 8 вузлів (верхня частина)

Всього: 16 невидимих вузлів що притягують людей!
